1. Download binary zenopanel
wget https://raw.githubusercontent.com/nextcore/zenopanel/main/dist/zenopanel-v0.0.9.tar.gz

2. Unzip
tar -xzvf zenopanel-v0.0.9.tar.gz

3. Jalankan 
./zeno

4. Akses aplikasi di port yg ditentukan di .env ( default: 3001 )
http://[IP_ADDRESS]:3001/login

5. Masuk ke Settings, Service Injector pilih Install
