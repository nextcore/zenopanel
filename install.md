1. Download binary zenopanel:
wget https://raw.githubusercontent.com/nextcore/zenopanel/main/dist/zenopanel-v0.2.0.tar.gz

2. Unzip & Masuk Folder:
tar -xzvf zenopanel-v0.2.0.tar.gz
cd zenopanel-v0.2.0

3. Salin Konfigurasi Environment
cp .env.example .env

4. Jalankan
./zeno

5. Akses aplikasi di port yg ditentukan di .env ( default: 3001 )
http://[IP_ADDRESS]:3001/login

6. Masuk ke Settings, Service Injector pilih Install
